# TIPE-Solar-tracker-
conception d'un suiveur solaire piloté par une carte Arduino. L'objectif est d'optimiser le rendement énergétique en orientant automatiquement un panneau photovoltaïque vers le soleil grâce à des capteurs LDR et un servomoteur MG995. Un algorithme ajuste l'angle en temps réel pour maximiser l'exposition lumineuse.
